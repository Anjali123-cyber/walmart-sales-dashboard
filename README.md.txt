🛒 Walmart Sales Dashboard

An interactive Power BI dashboard analyzing sales performance, customer behavior, and operational KPIs across Walmart branches. Created during my data analyst internship to derive actionable insights from retail data.

🔧 Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel (for data preprocessing)

🚀 Features
- Branch-wise and category-wise sales trends
- Monthly/quarterly performance visualization
- KPI cards (Total Revenue, Gross Income, etc.)
- Slicers for dynamic filtering

📁 Dataset Source
[Walmart Sales Dataset - Kaggle](https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales)
